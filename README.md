# breakout

A simple app to determine if a product will cause my girlfriend to breakout.

## Setup (PowerShell)

```powershell
pip install -r .\requirements.txt
```

## Run

```powershell
py .\main.py
>>> Product Name: {Desired Product Here}
```

## Security note

This code is for Gemini, so make your own API key through Google AI Studio and update your code accordingly.
